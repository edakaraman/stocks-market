<h1> Stock Market Data App With React </h1>
<p> Stock market data was retrieved with axios using finnHub API. </p>
<p> Structure/Technologies Used in the Project; </p>
<ul>
<li>Bootstrap </li>
<li> Axios</li>
<li>React-Icons </li>
<li> React Hooks</li>
<li> Context API</li>
<li>Local Storage </li>
<li> React Router </li>
<li> apexcharts (for graphics) </li>
  <li> Firebase </li> 
</ul>
